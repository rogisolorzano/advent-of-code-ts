## Advent of Code 2022

These are my [Advent of Code 2022](https://adventofcode.com/2022) solutions in Typescript.

### Getting started

Install dependencies

```
npm install
```

A `run` script is included at the root of the project to run the solutions by day.

Example:

```
./run 01
```
